# Crypto_Questions_and_Opinion_Analysis

This project focuses on analyzing questions and opinions related to cryptocurrency. It includes a web application built with Next.js for interactive analysis and visualization of crypto-related data.

## Features

- Interactive web interface for analyzing crypto questions and opinions
- Built with Next.js, React, and TypeScript
- Uses Radix UI components for modern UI
- Includes charting capabilities with Recharts

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies for the Next.js app:
   ```
   cd crypto-opinion-analyzer (1)
   npm install
   ```
4. Run the development server:
   ```
   npm run dev
   ```
